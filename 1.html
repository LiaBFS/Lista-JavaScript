Oque é JavaScript?

a. Trazer mais iteratividade para páginas web.

b.
- Navegador: o código é executado diretamente no navegador do usuário, é útil pra interações e feedback visual imediato.
- Servidor: o código é executado em um servidor web ou servidor de aplicação, útil pra acessar e manipular o banco de dados, além de processar lógicas de negócio.

c. É um ambiente isolado para testes e análise de malware. O sandbox serve para limitar o acesso do código JS no navegador, garantindo mais segurança sem prejudicar o sistema principal.

d. São objetos definidos no escopo global (aqueles que são acessíveis de qualquer lugar no código) e são criados pelo ambiente de execução.
